출결 관리기능과 정보공유 웹서비스

프로젝트 개요

문의 페이지 및 강사 전용 마이페이지 기능 개발을 중심으로 한 웹서비스입니다.

역할 및 기여

  • 프론트엔드 UI 구현
  • React 컴포넌트를 이용한 효율적 코드 작성
  • API 연동 및 상태관리(Zustand) 담당

트러블슈팅 및 문제 해결 과정

상태관리 오류 문제를 발견하고, Zustand 구조를 재설계하여 안정성을 높였습니다. 사용자 입력 데이터 유효성 검증 기능을 강화하였습니다.

기술 스택

React, React-query, Zustand, JavaScript, CSS
