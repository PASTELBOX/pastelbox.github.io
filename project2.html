프로젝트 요약

프로젝트명: 냉장고 관리웹 (2024.07.25 ~ 2024.09.20)
역할: Front-End Manager
기술 스택: React, React-query, Zustand, JavaScript, CSS
기여 내용:
주요 페이지 UI 설계 · 게시판/댓글/좋아요 기능 · 전역 상태관리 · 비동기 데이터 처리 · 협업 커뮤니케이션

담당 업무 및 주요 기여

  • 팀의 프론트엔드 매니저로서 전체 UI/UX 구조 기획과 코드컨벤션 제안
  • React 기반 커뮤니티 게시판, 좋아요/댓글 기능 설계 및 구현
  • 사용자 경험 개선을 위한 Zustand 글로벌 상태관리 도입
  • API 연동 및 비동기 처리 전문화, React-query로 효율적 데이터 흐름 관리
  • 팀원과의 실시간 협업(깃허브 pull-request 리뷰, 코드 합의 등)

주요 구현 예시

  • 프로젝트 구조와 모듈별 컴포넌트 분리, 유지보수성과 확장성을 고려함
  • 상태 관리(Likes, Comments, User Auth 등)는 Zustand 및 React-query로 간결하게 처리
  • 게시글 목록, 댓글 입력/수정, 좋아요 토글 기능을 직접 디자인 & 코드 작성
  • 실시간 알림, 구독 기능 등 사용자와 서버 간 인터랙션 UX 개선에 집중

협업 및 성장

다양한 역할의 팀원들과 MSA 환경에서 협업하며 구조적 접근의 중요성을 체감하였고, 프론트엔드-백엔드 API 설계 및 문서화, 깃허브 워크플로우와 실전 코드 리뷰를 통해 소통 역량 및 리더십을 키웠습니다.

성과 및 배운 점

  • 클라우드 기반 웹 개발, DevOps 환경 경험
  • 최신 프론트엔드 도구의 효율적 활용 및 코드 품질 향상 실전 체험
  • React-query/Zustand의 데이터 캐싱과 상태 동기화, 협업 커뮤니케이션 능력 성장
  • 실제 유저 중심 기능(좋아요, 댓글, 구독, 알림) 개발로 사용자 경험에 대한 시야 확장
